#SkullSpace on Freenode IRC network

Chat with #SkullSpace members and community members on the freenode IRC network. Web chat client.

As an alternative to web chat, there are also numerous IRC client programs you can install on your device. To connect you need to know is the server alias is chat.freenode.net and port 6667 (non-SSL) or port 6697 (SSL). Some chat clients come with the Freenode network built-in.
