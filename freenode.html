#SkullSpace on Freenode IRC network

Below is an embedded web chat client for the #SkullSpace channel on Freenode. You may also prefer a web chat client in a new tab or window.

As an alternative to web chat, there are also numerous IRC client programs you can install on your device. To connect you need to know the server alias is chat.freenode.net and port 6667 (non-SSL) or port 6697 (SSL). Some chat clients come with the Freenode network built-in.

We will relay chat messages to the presenters when they are looking for feedback and taking questions.

Please note the freenode general expectations for conduct and policies.
